Privacy Policy for Transcribe Notes and Note Auto-Copy Extension

Last updated: [INSERT DATE]

1. Overview

Transcribe Notes is a web-based tool for audio transcription and AI-assisted note generation. The Note Auto-Copy Extension is a companion Chrome extension for users of Transcribe Notes. This policy explains how the website and extension handle data.

2. What Transcribe Notes Does

Transcribe Notes allows users to record audio, transcribe speech, and generate draft notes using supported third-party AI providers selected by the user. Depending on configuration, audio, transcripts, prompts, and generated notes may be processed in the browser and may be sent to selected third-party AI providers in order to perform transcription and note generation.

3. User Responsibility

If Transcribe Notes is used with personal data, including special category data such as health data, the user is responsible for ensuring that use complies with applicable laws, regulations, and organizational requirements. This may include internal approvals, risk assessments, data processing agreements, transfer assessments, and other compliance measures depending on the user’s context.

4. Data the Website May Handle

Depending on how the service is used, the website may handle:

  • Audio recorded in the browser
  • Transcripts created from that audio
  • User-entered prompts and supplementary text
  • Generated notes
  • User-provided API keys entered into the app

5. How the Website Processes Data

  • Audio may be recorded through the browser.
  • Audio may be processed in browser memory and sent over secure connections to selected providers for transcription.
  • Transcripts and prompts may be sent over secure connections to selected providers for note generation.
  • Generated notes are returned to the browser.
  • API keys entered by the user may be stored locally in the browser as needed for app functionality.

6. Third-Party AI Providers

Transcribe Notes may interact with third-party AI and infrastructure providers supported by the app. Data handling terms, retention periods, geographic processing locations, and security terms may vary depending on the provider and configuration used. Users are responsible for reviewing and approving the provider setup they choose to use.

7. Note Auto-Copy Extension

The Note Auto-Copy Extension runs only on supported Transcribe Notes pages. When note generation finishes, the extension reads the completed note text from the page, copies that text to the user’s clipboard, and shows a browser or desktop notification.

  • The extension does not sell user data.
  • The extension does not transmit note content to the developer’s servers.
  • The extension does not use remote code.
  • The extension uses host permissions only on supported Transcribe Notes pages.
  • The extension uses clipboardWrite to copy the finished note to the clipboard.
  • The extension uses notifications to notify the user that copying is complete.
  • The extension uses the offscreen API to support reliable clipboard operations.

8. Data Sharing

The extension does not share copied note content with the developer or third parties. The website may send user content to the third-party provider selected for transcription or note generation because that is necessary for the service to function.

9. Data Retention

The extension is not intended to store copied note content beyond what is needed for immediate browser functionality. For the website, retention and processing may depend on the selected provider, browser behavior, and user configuration. Users should review the terms of the providers they use.

10. Security

Transcribe Notes is intended to use secure connections for communication with supported services. However, users remain responsible for evaluating their own compliance requirements, provider settings, and operational security needs.
